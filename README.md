# web-client
This repo stores the implementation for the RiskyTrees (https://beta.app.riskytrees.com) frontend (and risk model implementations).

## Running Locally

### Pre-Requisites

1. Ensure the RiskyTrees API service (https://github.com/riskytrees/api-service) is running locally.
1. Ensure you have `yarn` installed: https://yarnpkg.com/getting-started/install
1. From the root of this repo, run `yarn` to install all node dependencies

### Running
This is simple. Run `yarn run start` to start the frontend.
